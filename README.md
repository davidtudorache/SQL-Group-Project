# SQL-Group-Project
For this assignment I had to carry out an analysis of Travel and Tourism Ltd existing system (the details of the sytem where provided in the case study's documentation) and from this analysis produced a set of recommendations for its improvement and implementation. I presented the analysis and recommendations in the form of a report. The report contained an explaination the background, the purpose of the report, problem statement a Top-Down Entity Relationship Diagram (ERD) of the proposed system using UML diagramming conventions, Relational Data Analysis (RDA) and an explanation of any differences with the ERD previously generated. I implemented the database using data insertions and MariaDB.
